Intro
The file index.php implements a web service that will interact with the extranet in order to 
send or receive irbpeople's data.

How is it implemented?
The web service is declared on the server 'irbpeople-api.irbbarcelona.pcb.ub.es' which 
is a CNAME of 'irbsvr10.irbbarcelona.pcb.ub.es' (eservices)
